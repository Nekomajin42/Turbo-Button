Private window

If you want to use this extension in private mode, you have to activate in manualy in the Extension manager.

Open Extension manager

Shortcut

Alt+T is the default shortcut to enable/disable Turbo mode. You can change it in the Keyboard shortcuts dialog.

Open Keyboard shortcut manager

How to use

Two ways to enable/disable Turbo mode:

  • Click on the toolbar button!
  • Use the keyboard shortcut! (default: Alt+T)

Note

  • Please note, that once you have installed this extension, the default switch in the Settings won't work. This is how the extension API works.

Permissions

  • "notifications" to show you a little popup when the extension is updated.
  • "offroad" to access the Turbo setting.
